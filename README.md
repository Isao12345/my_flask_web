**Expense Tracker Dashboard**

โปรเจกต์นี้เป็นเว็บแอปพลิเคชันสำหรับบันทึกรายรับและรายจ่าย
พัฒนาด้วย Flask และ SQLite โดยมีหน้า Dashboard สำหรับแสดงสรุปข้อมูลทางการเงินของผู้ใช้

**การใช้งาน**

1. พิมพ์คำสั่ง (ใน terminal)

   cd my_flask_web

2. ติดตั้ง dependencies

   พิมพ์คำสั่ง (ใน terminal)

   pip install -r requiments.txt

3. รันเซิร์ฟเวอร์

   พิมพ์คำสั่ง (ใน terminal)

   python app.py

**วิธีการทำงานของระบบ**

ระบบรองรับการสมัครสมาชิกและเข้าสู่ระบบของผู้ใช้
ข้อมูลผู้ใช้ รายรับ รายจ่าย และหมวดหมู่ จะถูกจัดเก็บในฐานข้อมูล SQLite

เมื่อผู้ใช้เพิ่มรายรับหรือรายจ่าย ระบบจะบันทึกข้อมูลพร้อมวันที่
หน้า Dashboard จะนำข้อมูลทั้งหมดมาคำนวณยอดรวมรายรับ รายจ่าย และยอดคงเหลือ

ระบบจะจัดกลุ่มข้อมูลตามเดือน
และนำไปแสดงผลในรูปแบบกราฟแท่ง (Bar Chart) เพื่อช่วยให้ผู้ใช้เห็นภาพรวมทางการเงินได้ง่ายขึ้น

**ฟีเจอร์**

-สมัครสมาชิกและเข้าสู่ระบบ

-เพิ่ม / ดู รายรับ (Income)

-เพิ่ม / ดู รายจ่าย (Expense)

-จัดการหมวดหมู่ (Category)

-Dashboard แสดง

-ยอดรายรับรวม

-ยอดรายจ่ายรวม

-ยอดคงเหลือ

-กราฟรายรับและรายจ่ายรายเดือน

-เลือกดูข้อมูลตามเดือนที่ต้องการ
